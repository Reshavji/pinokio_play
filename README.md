====================================================
                PINOKIO PLAY 🎮
====================================================

PINOKIO PLAY is a Flutter + Firebase based Play &
Earn application with modern authentication and
reward features.

----------------------------------------------------
FEATURES
----------------------------------------------------

✔ Google Authentication
✔ Email & Password Login
✔ Firebase Authentication
✔ Firestore Database
✔ User Profile System
✔ Coins System
✔ Rewarded Ads Integration
✔ Banner Ads
✔ Modern Dark UI
✔ Error Handling
✔ Password Validation
✔ Responsive Design

----------------------------------------------------
TECH STACK
----------------------------------------------------

• Flutter
• Firebase Auth
• Cloud Firestore
• Google Sign-In
• Google Mobile Ads

----------------------------------------------------
INSTALLATION
----------------------------------------------------

1. Clone Repository

git clone <your-repository-url>

----------------------------------------------------

2. Open Project

cd pinokio_play

----------------------------------------------------

3. Install Dependencies

flutter pub get

----------------------------------------------------

4. Configure Firebase

flutterfire configure

----------------------------------------------------

5. Run Project

flutter run

----------------------------------------------------

6. Run On Chrome

flutter run -d chrome

----------------------------------------------------

7. Build Release APK

flutter build apk --release

APK Location:

build/app/outputs/flutter-apk/app-release.apk

----------------------------------------------------
FIREBASE SETUP
----------------------------------------------------

1. Create Firebase Project

2. Enable Authentication:
   • Google Sign-In
   • Email/Password

3. Create Firestore Database

4. Configure FlutterFire

----------------------------------------------------
COMMON ISSUES
----------------------------------------------------

1. Firebase Not Initialized

Run:
flutterfire configure

----------------------------------------------------

2. Missing firebase_options.dart

Run:
flutterfire configure

----------------------------------------------------

3. Google Login Not Working

• Enable Google Authentication
• Configure Firebase properly
• Add SHA Keys (Android)

----------------------------------------------------

4. AdMob Ads Not Loading

• Use correct Ad Unit IDs
• Initialize MobileAds properly
• Wait for ads to load
